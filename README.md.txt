# RERE – Recipe Recommender System

RERE is a C++ console-based recipe recommendation system developed as part of the Fundamentals of Computing course.

The application helps users manage recipes, maintain ingredient inventory, and receive recipe recommendations based on available ingredients.

## Features
- Recipe recommendation based on user ingredients
- Inventory-based recommendation
- Add new recipes
- Persistent history logging with timestamps
- Inventory management
- Categorized recipe storage
- Input validation

## Technologies & Concepts
- C++
- Object-Oriented Programming
- Dynamic Memory Allocation
- File Handling
- Arrays and Pointers
- String Parsing
- Algorithm Design
- Input Validation
- Modular Programming

## How to Run
```bash
g++ -std=c++17 -o RERE main.cpp
./RERE
```

## Project Structure
- `main.cpp` → source code
- `data/` → recipe and inventory files
- `documentation/` → project documentation
- `screenshots/` → program screenshots
